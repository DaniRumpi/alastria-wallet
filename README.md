# Alastria Prototype Ionic App

Example mobile application to implement Alastria user stories

## Installation

### Requirements
* Node.js 8+
* Yarn
* Ionic 3 / Angular 6
* Android / iOS SDK
* Cordova 7.1.0+
* Barcode Scanner

ionic cordova plugin add phonegap-plugin-barcodescanner

## Installation

```yarn
yarn install
```

## Development
```
$ ionic serve
```

## Android deployment
```
$ ionic cordova run android --device
```
